# Configuration

BLUX10K installs configuration files into your home directory with backups created before changes.

## Key locations

- `~/.zshrc` — shell configuration
- `~/.p10k.zsh` — Powerlevel10k prompt config
- `~/.config/neofetch/config.conf` — neofetch config
- `~/.config/starship.toml` — starship config
- `~/.config/private/env.zsh` — private environment variables

## Templates and examples

Update operations preserve your existing files and place new templates in:

- `~/.config/blux10k/examples/`
- `~/.config/blux10k/patches/`
